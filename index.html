Overview
Minecraft is a game about placing blocks and building anything you can imagine. At night, monsters come out, so be sure to build a shelter before that happens.

The game includes a full tutorial that we recommend following if this is your first time playing. It also has pop-up messages to help with your initial learning experience and, if you need help with something specific, there is a comprehensive section titled "How to Play," along with many customizable settings that you can find in the Help & Options menu.
